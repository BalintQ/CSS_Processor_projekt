Mire jók a CSS preprocesszorok?

Rövid leírás:

A CSS-preprocesszorok (előfeldolgozók) olyan szkriptnyelvek, amelyek kibővítik a hagyományos CSS képességeit, és olyan funkciókat adnak hozzá, mint a változók, beágyazott elemek, keverékek (mixinok) és matematikai műveletek, megkönnyítve a karbantarthatóbb, modulárisabb és hatékonyabb CSS írását. Ezeket a speciális szintaxisú kódokat egy fordító (compiler) átalakítja olvasható, szabványos CSS-re, amelyet aztán a böngészők feldolgozhatnak.
